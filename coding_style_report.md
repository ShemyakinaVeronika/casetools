## Описание написанного кода

Программа реализует решение квадратных уравнений вида ax² + bx + c = 0 с использованием графического интерфейса.

В программе реализованы следующие функции:
- quadratic_equation(a, b, c) — вычисляет корни уравнения;
- solve() — получает данные от пользователя и выводит результат;
- clear() — очищает поля ввода.

Программа состоит из двух основных частей:
- вычислительная логика;
- пользовательский интерфейс, реализованный с помощью библиотеки tkinter.

Для реализации использовались язык Python, библиотека tkinter и модуль math.

---

# Выбранный стиль программирования

В работе использован стандарт кодирования Python — **PEP 8** (Python Enhancement Proposal).

Основные принципы:

* отступы: 4 пробела;
* имена функций и переменных строчными буквами: snake_case;
* имена классов с большой буквы: UserProfile;
* пробелы вокруг операторов;
* импорты в начале файла;
* логическое разделение кода пустыми строками;
* ограничение длины строки (до 79 символов);
* аккуратное оформление многострочных конструкций.

# Автоматическое форматирование кода

Для форматирования использовался инструмент:

* black — автоматический форматировщик Python-кода.

Команда для запуска:

```bash
black main_after.py
```

Для анализа были использованы два файла:
