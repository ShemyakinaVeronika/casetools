# Отчет по работе с Git

## Общая информация

Целью работы является освоение основных возможностей системы контроля версий Git, включая создание репозитория, работу с ветками, добавление и изменение файлов, а также взаимодействие с удалённым репозиторием на GitHub.

---

В результате выполнения работы была сформирована следующая структура каталогов:


## Настройка репозитория

Инициализация локального репозитория:
```bash
git init
```

Подключение к удалённому репозиторию:
```bash
git remote add origin https://github.com/ShemyakinaVeronika/casetools.git
```

---

## Работа с ветками

Создание рабочей ветки:
```bash
git checkout -b Veronika
```
Просмотр списка веток:
```bash
git branch
```
Текущая ветка:
```bash
git branch --show-current
```
Переход между ветками:
```bash
git checkout master
git checkout Veronika
```

## Добавление файлов

Добавление файлов в индекс:
```bash
git add .
```
Добавление контретного файла в индекс:
```bash
git add <file>
```

Создание коммита:
```bash
git commit -m "message"
```
Отправка изменений на GitHub:
```bash
git push
git push --set-upstream origin Veronika
```

## Просмотр состояния и истории

Просмотр состояния:
```bash
git status
```
Просмотр истории коммитов:
```bash
git log
git log --oneline
```

Пример:
```bash
3f7354b (HEAD -> master, origin/master) Добавление метки и автора метки проекта в README
6453ec2 Изменение .gitignore
01afca7 (origin/Veronika, Veronika) Добавление .gitignore на github
92fecbc Удаление файла из проекта
1f7680d Добавление файла в проект
3b2c801 Исправление файла README.md
32ada2a Добавление проекта на github
```

Просмотр изменений в файлах:
```bash
git diff
```
